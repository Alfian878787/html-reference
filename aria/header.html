header – header # T

The header element represents the header of a section.

Content model #

header = (normal-character-data & common.elem.flow*) & header.attrs

Attribute model #

header.attrs = common.attrs & (common.attrs.aria.implicit.region | common.attrs.aria.landmark.banner)?

Element-specific attributes #

Assertions #

  • The “header” element must not appear as a descendant of the “footer” element.
  • The “header” element must not appear as a descendant of the “address” element.
  • The “header” element must not appear as a descendant of the “header” element.

Tag omission #

A header element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.flow

Details #

The header element typically contains the headings for a section (an h1-h6 element or hgroup element), along with content such as introductory material or navigational aids for the section.
