fieldset – set of related form controls # T

The fieldset element represents a set of form controls grouped under a common name.

Content model #

fieldset = legend?, (normal-character-data & common.elem.flow*) & fieldset.attrs

Attribute model #

fieldset.attrs = (common.attrs & common.attrs.aria.implicit.group?) & name? & disabled? & form?

Element-specific attributes #

Tag omission #

A fieldset element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.flow

Typical default display properties #

fieldset {
display: block;
margin-left: 2px;
margin-right: 2px;
padding: 0.35em 0.75em 0.625em;
border: 2px groove (internal value); }
