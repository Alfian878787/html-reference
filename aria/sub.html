sub – subscript # T

The sub element represents subscript.

Content model #

sub = (normal-character-data & common.elem.phrasing*) & sub.attrs

Attribute model #

sub.attrs = common.attrs

Tag omission #

A sub element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.phrasing

Typical default display properties #

sub {
vertical-align: sub;
font-size: smaller; }
