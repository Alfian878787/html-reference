abbr – abbreviation # T

The abbr element represents an abbreviation or acronym.

Content model #

abbr = (normal-character-data & common.elem.phrasing*) & abbr.attrs

Attribute model #

abbr.attrs = common.attrs & common.attrs.aria?

Element-specific attributes #

Tag omission #

An abbr element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.phrasing

Details #

The title attribute on the abbr element represents the expansion of the abbreviation or acronym.
