table – table # T

The table element represents a table; that is, data with more than one dimension.

Content model #

table = caption?, colgroup*, thead?, ((tfoot, (tbody* | tr+)) | ((tbody* | tr+), tfoot?)) & table.attrs

Attribute model #

table.attrs = common.attrs & summary? & common.attrs.aria?

Element-specific attributes #

summary = string
A prose description of the structure of the table.

Tag omission #

A table element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.flow

Typical default display properties #

table {
display: table;
border-collapse: separate;
border-spacing: 2px;
border-color: gray; }
