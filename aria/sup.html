sup – superscript # T

The sup element represents superscript.

Content model #

sup = (normal-character-data & common.elem.phrasing*) & sup.attrs

Attribute model #

sup.attrs = common.attrs

Tag omission #

A sup element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.phrasing

Typical default display properties #

sup {
vertical-align: super;
font-size: smaller; }
