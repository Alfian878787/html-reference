nav – group of navigational links # T

The nav element represents a section of a document that links to other documents or to parts within the document itself; that is, a section of navigation links.

Content model #

nav = (normal-character-data & common.elem.flow*) & nav.attrs

Attribute model #

nav.attrs = common.attrs & (common.attrs.aria.implicit.region | common.attrs.aria.landmark.navigation)?

Element-specific attributes #

Assertions #

  • The sectioning element “nav” must not appear as a descendant of the “footer” element.
  • The sectioning element “nav” must not appear as a descendant of the “address” element.

Tag omission #

A nav element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.flow

Typical default display properties #

nav {
display: block; }

Details #

Not all groups of links on a page need to be in a nav element — only sections that consist of primary navigation blocks are appropriate for the nav element. In particular, it is common for footers to have a list of links to various key parts of a site, but the footer element is more appropriate in such cases.
