dfn – defining instance # T

The dfn element represents the defining instance of a term.

Content model #

dfn = (normal-character-data & common.elem.phrasing*) & dfn.attrs

Attribute model #

dfn.attrs = common.attrs & common.attrs.aria?

Element-specific attributes #

Assertions #

  • The “dfn” element must not contain any nested “dfn” elements.

Tag omission #

A dfn element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.phrasing
