caption – table title # T

The caption element represents the title of the table that is its parent.

Content model #

caption = (normal-character-data & common.elem.phrasing*) & caption.attrs

Attribute model #

caption.attrs = common.attrs & common.attrs.aria.implicit.region?

Element-specific attributes #

Tag omission #

A caption element must have both a start tag and an end tag.

Permitted contexts #

  • table = caption?, colgroup*, thead?, ((tfoot, (tbody* | tr+)) | ((tbody* | tr+), tfoot?))

Typical default display properties #

caption {
display: table-caption;
text-align: center; }
