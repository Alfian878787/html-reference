output – result of a calculation in a form # T

The output element represents the result of a calculation.

Content model #

output = (normal-character-data & common.elem.phrasing*) & output.attrs

Attribute model #

output.attrs = common.attrs & name? & form? & for? & common.attrs.aria.implicit.region?

Element-specific attributes #

for = idrefs
Identifies one or more elements associated with the calculation whose result this output element represents.

Tag omission #

An output element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.phrasing
