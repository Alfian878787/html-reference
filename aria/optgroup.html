optgroup – group of options # T

The optgroup element represents a group of option elements with a common label.

Content model #

optgroup = option* & optgroup.attrs

Attribute model #

optgroup.attrs = common.attrs & label & disabled?

Element-specific attributes #

label = string
Specifies the name of the group of options its optgroup element represents.

Tag omission #

An optgroup element must have a start tag.

An optgroup element's end tag may be omitted if the optgroup element is immediately followed by another optgroup element, or if there is no more content in the parent element.

Permitted contexts #

  • select = optgroup* & option*
