br – line break # T

The br element represents a line break.

Content model #

br = empty & br.attrs

Attribute model #

br.attrs = common.attrs

Tag omission #

The br element is a void element. A br element must have a start tag but must not have an end tag.

Permitted contexts #

  • common.elem.phrasing
