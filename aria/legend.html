legend – title or explanatory caption # T

The legend element represents a title or explanatory caption for the rest of the contents of the legend element’s parent element.

Content model #

legend = (normal-character-data & common.elem.phrasing*) & legend.attrs

Attribute model #

legend.attrs = common.attrs & common.attrs.aria.implicit.region?

Element-specific attributes #

Tag omission #

A legend element must have both a start tag and an end tag.

Permitted contexts #

  • figure = (legend, (normal-character-data & common.elem.flow*)) | ((normal-character-data & common.elem.flow*), legend?)
  • fieldset = legend?, (normal-character-data & common.elem.flow*)
  • details = legend, (normal-character-data & common.elem.flow*)

Typical default display properties #

legend {
display: block;
padding-left: 2px;
padding-right: 2px;
border: none; }
