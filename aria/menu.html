menu – list of commands # T

The menu element represents a list of commands.

Content model #

menu = (li* | (normal-character-data & common.elem.flow*)) & menu.attrs

Attribute model #

menu.attrs = common.attrs & type? & label?

Element-specific attributes #

type = "toolbar" | "context"
The kind of menu being declared. The attribute has three states:
  • If the context keyword is specified, the element is declaring a context menu.
  • If the toolbar keyword is specified, the element is declaring a tool bar.
  • If the attribute is omitted, the element is a list of commands that is neither declaring a context menu nor defining a tool bar.
label = string
The label of the menu.

Assertions #

  • The element “menu” with the attribute “type=toolbar” must not appear as a descendant of the “a” element.
  • The element “menu” with the attribute “type=toolbar” must not appear as a descendant of the “button” element.

Tag omission #

A menu element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.flow

Typical default display properties #

menu {
display: block;
list-style-type: disc;
margin: 1em 0 1em 0;
padding-start: 40px; }
