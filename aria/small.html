small – small print # T

The small element represents so-called “small print” such as legal disclaimers and caveats.

Content model #

small = (normal-character-data & common.elem.phrasing*) & small.attrs

Attribute model #

small.attrs = common.attrs & common.attrs.aria?

Element-specific attributes #

Tag omission #

A small element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.phrasing

Typical default display properties #

small {
font-size: smaller; }
