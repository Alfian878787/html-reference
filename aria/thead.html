thead – table heading group # T

The thead element represents the block of rows that consist of the column labels (headings) for its parent table element.

Content model #

thead = tr* & thead.attrs

Attribute model #

thead.attrs = common.attrs

Tag omission #

A thead element must have a start tag.

A thead element's end tag may be omitted if the thead element is immediately followed by a tbody or tfoot element.

Permitted contexts #

  • table = caption?, colgroup*, thead?, ((tfoot, (tbody* | tr+)) | ((tbody* | tr+), tfoot?))

Typical default display properties #

thead {
display: table-header-group;
vertical-align: middle;
border-color: inherit; }
