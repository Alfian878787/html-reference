ruby – ruby annotation # T

The ruby element allows spans of phrasing content to be marked with ruby annotations.

Content model #

ruby = ((normal-character-data & common.elem.phrasing*), (rt | (rp, rt, rp)))+ & ruby.attrs

Attribute model #

ruby.attrs = common.attrs & common.attrs.aria?

Element-specific attributes #

Tag omission #

A ruby element must have both a start tag and an end tag.

Permitted contexts #

  • common.elem.phrasing
